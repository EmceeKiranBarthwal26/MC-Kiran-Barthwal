# Kiran Barthwal - Professional Event Host & MC

Welcome to the source code repository for Kiran Barthwal's professional portfolio website.

## 🌐 Live Website
The website is currently hosted live via GitHub Pages. You can view it here:
**[https://EmceeKiranBarthwal26.github.io/MC-Kiran-Barthwal/](https://EmceeKiranBarthwal26.github.io/MC-Kiran-Barthwal/)**

## 📂 Project Structure
This is a lightweight, static HTML/CSS/JS website. It does not require a backend server or database to run.

- `index.html`: The main structural file containing the landing page, gallery definitions, about sections, and contact form structure.
- `styles.css`: The styling file responsible for the responsive layout, animations, gold-and-dark theme, and overall UI/UX.
- `script.js`: Handles interactivity, such as the mobile navigation menu, animated counters, smooth scrolling, and gallery filtering/lightboxes.
- `Media/`: Contains all the local images used on the website. (Note: File names use underscores instead of spaces to ensure 100% web compatibility).

## 🚀 Deployment
This website is automatically deployed using **GitHub Pages**. 
- **Source Branch:** `main`
- **Folder:** `/ (root)`

Any changes pushed to the `main` branch (specifically modifying the root files) will automatically trigger a rebuild and update the live website within 1-2 minutes.

## 💻 Local Development
To run this project locally, you don't need any complex tools:
1. Clone the repository.
2. Open `index.html` directly in your web browser (Chrome, Safari, etc.).
3. Optionally, use a local server like VS Code's "Live Server" extension for hot-reloading.
